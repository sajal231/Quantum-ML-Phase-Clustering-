# Quantum-ML-Phase-Clustering

This repository contains an unsupervised machine learning analysis 
on spin system data using clustering methods (K-means, Gaussian Mixture Models) 
and dimensionality reduction techniques (PCA, t-SNE).

## Project Overview
- Input: Spin coupling parameters (Jx, Jy, Jz)
- Output: Cluster labels and visualization of quantum phases
- Techniques: PCA, t-SNE, K-means, Gaussian Mixture Models
- Cluster evaluation: Silhouette score, purity

## Features
- Phase encoding & standard scaling
- PCA + t-SNE projections
- Best K selection using Silhouette analysis
- Cluster visualization with K-means and GMM
- Cluster purity evaluation

## Applications
- Quantum Phase Classification
- Condensed Matter Physics
- Exploratory Data Analysis for Quantum Systems
